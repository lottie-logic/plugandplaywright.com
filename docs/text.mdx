# Text

## `getText`

Check if a text content is visible on the page, identified by the string.

<TabsBasic
  Tab1Label="Utility Function"
  Tab1Content={`await getText(page, <TEXT CONTENT>)`}
  Tab2Label="Playwright"
  Tab2Content={`await expect(page.getByText(<TEXT CONTENT>, { exact: true })).toBeVisible();`}
  Tab3Label="Parameters"
  Tab3Content={"string | number"}
/>

:::tip

Make sure the string / number you add is **exact** (including correct case) otherwise this test will fail.

Use [getTextPartialMatch](./button/#getTextPartialMatch) for a more leniant selector

:::

```javascript {4}
import { getText } from "plugandplaywright";

export async function layoutCheck(page) {
  await getText(page, "Text content to check");
}
```

## `clickText`

Click a piece of text that is visible on the page, identified by the text content.

<TabsBasic
  Tab1Label="Utility Function"
  Tab1Content={`await clickText(page, <TEXT CONTENT>)`}
  Tab2Label="Playwright"
  Tab2Content={`await page.getByText(<TEXT CONTENT>, { exact: true }).click(); 
await page.waitForLoadState("load");`}
  Tab3Label="Parameters"
  Tab3Content={"string | number"}
/>

Not only click the text but auto wait for the load to complete.

This means the test will wait until the page is ready for the next action.

:::tip

Make sure the string / number you add is **exact** (including correct case) otherwise this test will fail.

Use [clickTextPartialMatch](./button/#clickTextPartialMatch) for a more leniant selector

:::

```javascript {4}
import { clickText } from "plugandplaywright";

export async function layoutCheck(page) {
  await clickText(page, "Text Content to CLick");
}
```
