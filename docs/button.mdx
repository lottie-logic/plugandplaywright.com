# Button

## `getButton`

Check if a button is visible on the page, identified by the text content.

<TabsBasic
  Tab1Label="Utility Function"
  Tab1Content={`await getButton(page, <BUTTON TEXT>)`}
  Tab2Label="Playwright"
  Tab2Content={`await expect(page.getByRole("button", { name: <BUTTON TEXT>, exact: true })).toBeVisible();`}
  Tab3Label="Parameters"
  Tab3Content={"string | number"}
/>

:::tip

Make sure the string / number you add is **exact** (including correct case) otherwise this test will fail.

Use [getButtonPartialMatch](./button/#getButtonPartialMatch) for a more leniant selector

:::

```javascript {4}
import { getButton } from "plugandplaywright";

export async function layoutCheck(page) {
  await getButton(page, "Text of your button");
}
```

## `clickButton`

Click a button that is visible on the page, identified by the text content.

<TabsBasic
  Tab1Label="Utility Function"
  Tab1Content={`await clickButton(page, <BUTTON TEXT>)`}
  Tab2Label="Playwright"
  Tab2Content={`await page.getByRole("button", { name: <BUTTON TEXT>, exact: true }).click(); 
await page.waitForLoadState("load");`}
  Tab3Label="Parameters"
  Tab3Content={"string | number"}
/>

Not only click the button but auto wait for the load to complete.

This means the test will wait until the page is ready for the next action.

:::tip

Make sure the string / number you add is **exact** (including correct case) otherwise this test will fail.

Use [clickButtonPartialMatch](./button/#clickButtonPartialMatch) for a more leniant selector

:::

```javascript {4}
import { clickButton } from "plugandplaywright";

export async function layoutCheck(page) {
  await clickButton(page, "Text of your button");
}
```
